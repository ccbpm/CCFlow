标题
显示在窗口的文字，比如:请选择采购产品.
弹窗属性 高度: 宽度: 用于控制弹出窗口的大小 设置显示列数 列
数据源类型 选择类型
分组数据源
  • 设置一个可以返回json的数据源该数据源有No,Name,ParentNo三个约定的列.
  • SQL事例1: SELECT No,Name FROM Demo_BanJi
  • SQL事例2: SELECT No,Name FROM Port_Dept
  • Url事例: /DataUser/Handler.ashx?DoType=ReqDepts
实体数据源
  • 设置一个可以返回json的数据源该数据源有No,Name, 关联外键列 三个约定的列.
  • 比如:Url事例: /DataUser/Handler.ashx?DoType=Demo_Students
  • SQL事例: SELECT No,Name,FK_BanJi FROM Demo_Student
  • SQL事例: SELECT No,Name,FK_Dept FROM Port_Emp
确定后执行的JS
  • 该选项可以为空,弹出框确定后执行的JS，可以直接写方法名或者方法().
