分组列表平铺
基本信息
标题
显示在窗口的文字，比如:请选择采购产品.
弹窗属性 高度: 宽度: 用于控制弹出窗口的大小
数据源类型 选择类型
分组数据源
  • 设置一个可以返回json的数据源该数据源有No,Name,ParentNo三个约定的列.
  • SQL事例: SELECT No,Name FROM Demo_BanJi
  • Url事例: /DataUser/Handler.ashx?DoType=ReqDepts
实体数据源
  • 设置一个可以返回json的数据源该数据源有No,Name, 关联外键列 三个约定的列.
  • 比如:Url事例: /DataUser/Handler.ashx?DoType=Demo_Students
  • SQL事例: SELECT No,Name,FK_BanJi FROM Demo_Student
返回值填充其他数据控件的数据源
  • 该数据源可以为空
  • 如果配置了，系统就会按照配置数据源所返回的列与字段名相同的控件匹配进行赋值。
  • 数据源必须包含@Key变量，并且当前模式为单选。
  • 例如：SELECT No AS YuanGongBianHao, Name AS MingCheng, FK_BanJi AS DianHua FROM Demo_Student WHERE No='@Key'
