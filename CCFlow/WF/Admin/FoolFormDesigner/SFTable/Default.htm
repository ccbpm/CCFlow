外键表
  • BP类(外键表)
  • 创建本地数据表(外键表)
  • 把现有的表或者视图作为外键表注册到表单数据字典.
外部数据源
  • SQL查询表
  • WebServices
  • 微服务Handler外部数据源,(需要重写\DataUser\SFTableHandler.ashx 文件.)
  • JavaScript外部数据源(需要重写\DataUser\SFTable.js 文件.)
帮助
  • 外键表再组成SQL的时候，需要关联查询，创建一个字段，名称是关联 join 查询出来的。
  • 外部数据源创建2个字段比如：一个是abc，另外一个是abcT, abcT字段存储名称, abc字段存储编号列。
