流程报表定义说明
  • 一个流程有三种报表，分别是:我发起的流程， 本部门发起的流程，高级查询。
  • 每个报表都对应：查询组件、分析组件、高级查询组件、对比分析组件、交叉报表组件.
  • 每一种报表都有自己的控制权限、查询条件、显示内容的自定义。
  • 点击左边进行为每一种报表定义权限。
我的发起的流程
  • 该报表仅仅显示我发起的流程.
  • 你可以在报表定义工具里定义查询方式、显示内容.
  • 默认所有人都有权限.
我参与的
  • 该报表仅仅显示我发起的流程.
  • 你可以在报表定义工具里定义查询方式、显示内容.
  • 默认所有人都有权限.
我部门发起的流程
  • 该报表仅仅显示我部门发起的流程.
  • 你可以在报表定义工具里定义查询方式、显示内容.
  • 定义权限.
所有部门发起的
  • 可以查询所有部门发起的流程.
  • 你可以在报表定义工具里定义查询方式、显示内容.
  • 定义权限.
