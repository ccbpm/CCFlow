
说明;
============================================

1，该目录是存放excel表单模版的目录，excel表单模版必须是.xls格式。

2, 命名格式为 节点表单编号+.xls ，比如表单编号为MySheet 那末文件是 MySheet.xls 

3, 在表单绑定到节点后，有的时候会遇到，一个流程中的多个节点同时绑定一个excl表单的情况，但是每个表单的显示内容又不尽相同，为了解决此问题，我们允许了一个excel表单有可以绑定多个多个excel文件模版，但是必须有一个是默认的表单模版。
   在流程为每个节点绑定表单的时候，就会让设计人员选择一个表单模版文件。
  
   以上情况，ccbpm 不在提供界面维护，文件模版的文件名必须以表单的编号作为开头，命名放入到这个目录下。
    比如； 
    MySheet.xls   默认的表单
    MySheetxxxxxxxxxxxxx.xls  新增加的表单。
    MySheetyyyyyy.xls
